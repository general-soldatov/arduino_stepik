Светодиоды можно включать и выключать не только с помощью функции digitalWrite(), но и регулировать яркость свечения с помощью широтно-импульсной модуляции.

Оценка видимой яркости света может производиться с помощью так называемого отношения времени, когда ток включён, ко времени одного цикла ШИМ - это коэффициент заполнения.

Для плат Arduino использовать ШИМ можно только на цифровых входах 3, 5, 6, 9, 10 и 11. Они отмечены символом тильды (~).

**Вова**: Смотри, кажется это то, что нам надо:

Создать сигнал ШИМ можно с помощью функции analogWrite(a, b), которая принимает два аргумента:
a - номер цифрового выхода
b - значение коэффициента заполнения от 0 до 255.

При этом самый большой коэффициент заполнения в 100% будет соответствовать числу 255, а 0 - 0%.
