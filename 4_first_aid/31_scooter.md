Для считывания значения кнопки, нам необходимо использовать функцию digitalRead(3), где 3 - это номер цифрового вывода. Эта функция возвращает значение HIGH - если ток подаётся на наш вывод, а иначе LOW.

Подача тока осуществляется всегда в схемах с подтягивающим резистором. То есть при опущенного кнопке digitalRead будет возвращать HIGH. Когда мы нажимаем на кнопку, ток начинает идти с подтягивающего резистора на GND, поэтому, когда кнопка нажата, digitalRead будет возвращать LOW.

Прочитав описание инструкции программирования кнопки, Маша перешла к написанию кода. Получилась такая конструкция в функции loop():

```cpp
void loop() {
  if (digitalRead(3) == LOW) {
    for (int i = 0; i < 3; i++) {
      digitalWrite(4, HIGH);
      delay(500);
      digitalWrite(4, LOW);
      delay(500);
    }
  }
}
```
Тем самым при нажатии на кнопку поворотника цикл моргания будет осуществляться в течение трёх раз.
