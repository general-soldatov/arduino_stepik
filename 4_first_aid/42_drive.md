На заре ребята поднялись и после завтрака принялись за работу.

**Вова**: Для ближнего света и кнопка сломалась. Что делать?

**Маша**: Давай я попробую запрограммировать включение света через Ардуино!

**Вова**: Но новый код не будет мешать программе поворотников?

**Маша**: Нет, у меня есть идея как это реализовать.

**Вова**: Только лампу придётся подключать через реле! Наша плата имеет рабочее напряжение 5 В, а проводка - 12 В. Иначё всё может погореть.

**Маша**: Хорошо, я поищу информацию, как его подключать.

**Вова**: Отлично, а я тогда попробую восстановить проводку к стоп сигналу. На рукоять тормоза должен идти концевик (типа кнопка), которая и подаёт питание на стоп сигнал.

После обсуждения ребята приступили к ремонту скутера. Маша нашла картинку релейного модуля и информацию к его подключению.

Информация по выводам:

VCC - питание реле (+ 5 В)
GND - питание реле (- 5 В)
IN - входной сигнал от микроконтроллера
NC - разомкнут по умолчанию (normally closed)
COM - общий контакт реле (common pin)
NO - замкнут по умолчанию (normally open)

**Маша**: Хмм, значит, если я подаю сигнал с микроконтроллера на вывод IN, то NC будет замыкаться с общим контактом. Попробую реализовать это в коде.

В новом скетче Маша написала следующий код:
```cpp
void setup() {
  pinMode(9, OUTPUT);
  pinMode(10, INPUT_PULLUP);
}
void loop() {
  if (digitalRead(10) == LOW) {
    digitalWrite(9, HIGH);
  }
  else {
    digitalWrite(9, LOW);
  }
}
```
При нажатии кнопки реле замыкалось, при отпускании - размыкалось.
