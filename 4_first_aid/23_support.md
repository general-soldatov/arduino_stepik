Пока Володя изучал проводку скутера, Маша открыла раздел подключения кнопок.

Подключение кнопки осуществляется таким образом: один вывод кнопки идёт на цифровой вывод платы, а второй - на GND (минусовая клемма).

Есть более сложные схемы с подтягивающим резистором, однако касаться мы их пока не будем, поскольку подтягивающий резистор можно подключить программным способом. Для этого в функции setup пропишем следующую функцию настройки вывода:
```cpp
void setup() {
  pinMode(3, INPUT_PULLUP);
}
```
Режим `INPUT_PULLUP` позволяет настроить вывод платы на ввод, а также подключить подтягивающий резистор.

Также для обработки нажатия кнопки, нам необходимо коснуться темы условного оператора. Синтаксис условного оператора имеет вид:
```cpp
if (логическое выражение) {
// блок кода, выполняющийся в случае истинности логического выражения
}
else {
// блок кода, выполняющийся в случае, когда логическое выражение ложно
}
```
