## Начало обучения

**Артём**: Итак, ребята, вы наверняка знаете, что где пишется код для платы Arduino, а также какие главные функции входят в скетч?

**Маша**: Да, конечно, это программа Arduino IDE. А две главные функции - `setup()`, где мы настраиваем программу и этот код повторяется один раз. А вторая - `loop()` - она выполняется уже бесконечное количество раз до непосредственного выключения питания.

**Артём**: Правильно, Маша! Теперь давайте взглянем на саму плату Arduino UNO, вверху платы мы видим нумерованные выводы от 0 до 13. Это и есть управляемые выводы платы, с помощью которых мы можем управлять любыми подключенными устройствами. Например, светодиодами.

**Маша**: А как с помощью кода можно зажечь светодиоды?

**Артём**: Для начала мы переходим в функцию `setup()` и пишем следующий код:

```cpp
void setup() {
    pinMode(0, OUTPUT);
}
```

Таким образом мы настраиваем нулевой пин микроконтроллера на выход. Затем переходим в функцию `loop()` и добавляем туда команды:
```cpp
void loop() {
    digitalWrite(0, HIGH);
    delay(100);
}
```

Здесь с помощью функции `digitalWrite` мы подаём на нулевой пин высокое напряжение (в районе 5 В), а функция delay позволяет над приостанавливать работу микроконтроллера на то количество миллисекунд, которое передадим. В данном случае будет 100 миллисекунд.
