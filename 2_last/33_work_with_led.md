**Вова**: А если такой же фокус сделать со временем? Только измени время задержки, чтобы мы понимали, что прошивка девайса прошла успешно!

Маша попробовала переписать код и для времени:
```cpp
#define LED_BUILTIN 10
#define PAUSE_MS 500

void setup() {
    pinMode(LED_BUILTIN , OUTPUT);
}

void loop() {
    digitalWrite(LED_BUILTIN , HIGH);
    delay(PAUSE_MS );
    digitalWrite(LED_BUILTIN , LOW);
    delay(PAUSE_MS );
}
```

После успешной компиляции кода и перепрошивки девайса светодиод заморгал гораздо быстрее.
