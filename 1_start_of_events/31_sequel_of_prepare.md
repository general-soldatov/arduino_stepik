# Урок 3. Продолжаем подготовку

**Маша:** А теперь давай с тобой скачаем софт для Arduino. Для этого выполним следующие шаги:

✅Заходим на сайт https://www.arduino.cc/ (возможно понадобится VPN)

✅Во вкладке "Products" выбираем "Arduino IDE"

✅На текущей странице уже выбираем версию ArduinoIDE и скачиваем для своей операционной системы.
