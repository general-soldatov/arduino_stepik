Ваш микроконтроллер сможет петь, пищать, сигналить и даже проигрывать мелодии! Сделать это можно с помощью функцию `tone()` — голос вашего Arduino.

`tone()` — это встроенная функция Arduino, которая позволяет генерировать меандр (прямоугольный сигнал) заданной частоты на указанном цифровом пине. Если подключить к этому пину пьезодинамик или маленький динамик, вы услышите звук!

```cpp
tone(pin, frequency);
tone(pin, frequency, duration);
```

`pin` - Номер цифрового пина, к которому подключён динамик
`frequency` - Частота звука в герцах (Гц)
`duration` - Длительность звучания в миллисекундах (необязательный параметр)

Как это звучит? Рассмотрим небольшую таблицу частот:

Нота	Частота (Гц)	Ощущение
До (C4)	262	Низко, спокойно
Ре (D4)	294	Немного выше
Ми (E4)	330	Светло
Фа (F4)	349	Чуть тревожно
Соль (G4)	392	Бодро
Ля (A4)	440	Эталон, строй
Си (B4)	494	Ярко, устремлённо

Так вот, если подать частоту 440 Гц, динамик колеблется 440 раз в секунду — и рождается нота Ля, на которой настраивают оркестры.

Давайте напишем простейший пример музыкальной мелодии:
```cpp
int speakerPin = 8;

void setup() {
    // Пин настраивать не нужно — tone() сама переключает его в OUTPUT
}

void loop() {
    tone(speakerPin, 440);  // Включаем ноту Ля
    delay(1000);             // Играем 1 секунду
    noTone(speakerPin);      // Выключаем звук
    delay(500);              // Пауза полсекунды
}
```
⚠️ Важно: `tone()` не блокирует программу. Пока звучит одна нота, Arduino может делать другие дела. Но если не вызвать `noTone()` или не указать `duration`, звук будет играть бесконечно (пока к плате подаётся ток).
Функция `noTone(pin)` останавливает генерацию сигнала на указанном пине. Если вы запустили `tone()` без длительности, то без `noTone()` динамик будет играть безостановочно.

```cpp
tone(8, 1000);    // Писк на 1 кГц
delay(2000);
noTone(8);        // Тишина
```
